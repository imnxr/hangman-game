## Short Portfolio Description

This project is a responsive browser-based Hangman game built with HTML, CSS, and JavaScript. It expands a classic word game into a 100-level experience with progressive difficulty, clue letters, unlockable hints, and a clean SVG hangman scene. The project was structured with separate files and designed to feel like a complete front-end product rather than a small demo.
